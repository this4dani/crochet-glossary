From Curious Student to Pattern API

It started in middle school when a teacher introduced me to computers. Something just clicked. By high school, I was playing with Python and random number games, fascinated by how code could make computers respond.

Then came the MySpace era. I had to have the coolest page, which meant learning HTML and becoming obsessed with hex codes. Little did I know that obsession would lead to building tools for crochet patterns decades later.

I was a hands-on learner who learned best by doing, so I taught myself everything through trial and error. Turns out, being curious and stubborn about figuring things out is exactly what's needed to solve real-world problems.

Building the Future of Crochet

The Problem

After 30 years of crocheting, I still spend hours rewriting entire patterns just to make simple adjustments. Different yarn weight? Rewrite everything. Gauge slightly off? Start from scratch.

The Solution

I'm building a pattern adjustment API that automates the math, so crocheters can focus on creating instead of calculating. Because our time should be spent making beautiful things, not rewriting patterns.

The Heart

Serving church communities with hands-on and technology solutions while building tools that make crochet more accessible for everyone. Life is a hot mess, but I have Joy in the Lord.
